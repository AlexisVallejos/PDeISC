Terminos y Condiciones

Resumen del registro.
Este formulario es una practica academica inspirada en un alta de socios. Los datos cargados se usan solo dentro de esta sesion y no reemplazan un proceso oficial del club.

Uso responsable.
El usuario debe ingresar informacion consistente, revisar cada campo y aceptar que el sistema valida DNI, telefono, email, nacionalidad, usuario y contrasena antes de confirmar el alta.

Privacidad.
Los registros se mantienen en memoria para fines demostrativos del ejercicio. No se persisten en una base externa ni generan efectos reales sobre una cuenta oficial.

Alcance de la demo.
El envio de correo es de prueba. Si el servicio externo no responde, el sistema mantiene el alta local para que la funcionalidad principal del ejercicio siga operativa.

Fin del resumen de terminos.
