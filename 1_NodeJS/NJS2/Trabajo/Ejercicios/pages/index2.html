Módulos http y fs de Node.js

¿Cómo funciona este servidor?

Este archivo HTML fue leído del disco usando el módulo fs y entregado al navegador mediante el módulo http.

El módulo fileLoader encapsula toda la lógica de lectura:

  • El servidor recibe la petición HTTP
  • fileLoader.leerArchivo() lee el archivo del disco
  • El contenido binario se envía como respuesta con el Content-Type correcto
