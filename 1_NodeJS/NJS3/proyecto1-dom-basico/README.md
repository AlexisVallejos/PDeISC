# Proyecto 1

## Objetivo

Practicar manipulacion basica del DOM con botones.

## Que hace

- Agrega un `H1` con el texto `Hola DOM`.
- Cambia el texto del `H1` a `Chau DOM`.
- Cambia el color del `H1`.
- Agrega una imagen.
- Cambia la imagen.
- Cambia el tamano de la imagen.

## Archivos principales

- `app.js`: servidor Express en el puerto `3001`.
- `public/index.html`: contiene los botones y el area de trabajo.
- `public/css/style.css`: estilos visuales del proyecto.
- `public/js/script.js`: maneja los eventos de los botones.

## Funcionamiento

El script busca los elementos principales de la pagina, crea nodos si no existen y actualiza el contenido cada vez que el usuario presiona un boton.
