# Proyecto 4

- Crea 5 enlaces.
- Modifica el atributo `href` de cada enlace.
- Cambia el texto visible de cada nodo.
- Registra en pantalla cada cambio realizado.

## Archivos principales

- `app.js`: servidor Express en el puerto `3004`.
- `public/index.html`: estructura de botones, enlaces y registro.
- `public/css/style.css`: estilos de paneles y listas.
- `public/js/script.js`: crea nodos y cambia atributos.

## Funcionamiento

Se usan arreglos con datos originales y nuevos datos. Primero se crean los enlaces. Despues se reemplaza el valor de `href` y se informa el cambio en una lista.
