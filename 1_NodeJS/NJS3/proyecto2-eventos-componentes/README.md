# Proyecto 2

## Objetivo

Mostrar distintos componentes con diferentes eventos de JavaScript.

## Que hace

- Tiene navegacion entre componentes.
- Usa eventos `click`, `mouseenter`, `mouseleave`, `dblclick`, `keydown` y `change`.
- Cambia contenido o estilo segun la accion del usuario.

## Archivos principales

- `app.js`: servidor Express en el puerto `3002`.
- `public/index.html`: estructura de la navegacion y los paneles.
- `public/css/style.css`: diseno y distribucion del contenido.
- `public/js/script.js`: controla las pestañas y eventos.

## Funcionamiento

El usuario selecciona un componente y el script activa solo esa seccion. Dentro de cada panel se ejecuta un evento distinto para mostrar el comportamiento del navegador.
