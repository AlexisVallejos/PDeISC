Formulario con axios.post() y fetch POST

Completa nombre y email. Tienes dos paneles separados para enviar los datos de dos formas distintas y ver el ID de respuesta.

Volver al launcher

Dos formas de enviar el formulario

Cada panel valida nombre y email antes de enviar. El email debe tener un formato correcto y no permite puntos consecutivos.

Panel Axios

Enviar con axios.post()

Esta opcion usa Axios para hacer el POST al backend y mostrar el ID que responde la API.

Panel Fetch

Enviar con fetch POST

Esta opcion usa fetch con metodo POST y cuerpo JSON para enviar los mismos datos y mostrar el ID de respuesta.
